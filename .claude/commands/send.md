# Send Bot Message

Send a message to the Tests channel using the testbot account.

## Usage

```
/send Hello world!
```

## Instructions

Run the send-bot-message script with "Tests" as the room and the user's arguments as the message:

```bash
/Users/mickaelfm/projects/perso/organizer/server/send-bot-message.sh "Tests" "$ARGUMENTS"
```

Execute this command and report the result to the user.
